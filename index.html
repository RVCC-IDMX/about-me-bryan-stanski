Image Hovering Effects and their Great Uses

The hover effects will not work on this page if you are on mobile

While there are a vast number of ways to design and style your web pages, the one I will be focusing on telling you about here is the Image Hover Effects css styling. Before I jump into how Image Hover Effects work, we should probably talk about why you should even care about them and what they are actually used for.

Well, for starters Image Hover Effects can include a wide-variety of effects including such as enlarging the image, adding color or filters to the image, bringing up information via a text overlay and so much more. By utilizing these effects, you can take any image that you have on your site and increase engagment with your audience as well as increase sales and overall hint towards what you want your audience to know they can do on your site. All of that is achieved by drawing the user's attention to a specific part of the website and providing the necessary information they need at any given point in time.

When it comes to creating these hover effects, there is a very easy and quick way to create them at a basic level. Even though there are many different hover effects you can do, for this example I will be showing you how to add a text overlay to your images To start off you will need to import your image into your site and go into/create a css document and begin styling your image to the size and position that you will want it on your site. Next you will need to make the following:

  1. Make sure you have a class name for your image (for this example we will be using: example_image)
    2. Put a div around your image and text with the class name of container (container is used since it is easy to identify)
  2. You will also need a class name for both a "overlay" div and a "text" div (with those class names respectively)
  3. Finally, you will need to style your .text div to your liking (in this case it will be centered on the image)

After you have done that you will using the .overlay div to create that background to your hover effect like so:

.overlay { position: absolute; top: 0; bottom: 0; left: 0; right: 0; height: 100%; width: 100%; opacity: 0; transition: .5s ease; background-color: #008CBA; }

And finally once you have all that setup, you only need to add the actual hover effect to your css file with:

.container:hover .overlay { opacity: 1; }

Now by doing all of that, you should have a hover effect that looks something like the one at the end of this article.

So while there are many uses for image hover effects, there is one downside to them and that is that it only works on devices that have the ability to hover over your images (like desktops/laptops). This means mobile devices do not get to interact with these effects, so when you design your webpages with hover effects, make sure you keep that in mind and create an alternative for mobile devices (which you can see below in my example).
